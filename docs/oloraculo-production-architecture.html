Rules

Polytrade

Read docs/code/tests only. Extract ideas, not runtime dependencies.

AWS

Use a clean Oloraculo-owned stack with SSM ops. Measure p50/p90/p99 before latency claims.

Storage

Local disk is hot cache only. History goes to R2/S3-compatible object storage.

Execution

No live order path until Oloraculo has its own approval, WAL, reconciliation, and risk gates.

Core scalp formula

entry side + hedge side < 1.00
profit/share = 1 - entry - hedge
roi = profit / (entry + hedge)
required_hedge = 1 / (1 + target_roi) - entry
