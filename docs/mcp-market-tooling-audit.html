Do not install now

The marketplace is packed with high-ranking but irrelevant tools. High popularity does not equal fit. The trading system needs fewer tools with stronger gates.
Class / examples reviewedDecisionWhy
Polymarket / trading MCPs: Polymarket, QuantDinger, TradingView, AI Trader, Maverick reject They either advertise autonomous trade/position features or generic market analysis. NativePM money path must stay Rust-first with exact approval/WAL/risk gates. Build a custom read-only PM market importer instead.

News/trend aggregators: TrendRadar, OpenNews, Zenfeed reject for trading Tempting but noisy. For football availability/news, use explicit source fetchers and evidence classifications inside Oloraculo.
